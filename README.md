# Unec İmtahan Materialları

Məzmun və məqsəd
Bu sayt tələbələr üçün imtahan materiallarını bir yerdə toplamaq məqsədi daşıyır. Sayt rəsmi qurum deyil və heç bir universitetlə əlaqəsi yoxdur.

PDF materiallar
Saytda yerləşdirilən PDF fayllar istifadəçilər tərəfindən paylaşılmış və ya açıq mənbələrdən götürülmüşdür. Bütün hüquqlar orijinal müəlliflərə aiddir.

Məsuliyyətin məhdudlaşdırılması
Saytda yerləşdirilən məlumatların dəqiqliyinə tam zəmanət verilmir. İstifadə zamanı yaranan hər hansı zərərə görə sayt sahibi məsuliyyət daşımır.

Üçüncü tərəf xidmətlər
Saytda bəzi üçüncü tərəf xidmətlər istifadə olunur, o cümlədən Formspree və Google Fonts. Bu xidmətlər öz qaydalarına əsasən işləyir.

Dəyişikliklər
Bu şərtlər istənilən vaxt yenilənə bilər. Yenilənmiş versiya bu səhifədə paylaşılır.
